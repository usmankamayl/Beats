## 🛠️ Технологии и подходы
 
| Технология / Подход | Описание |
|---|---|
| **HTML5 / CSS3** | Семантическая разметка, современные стили |
| **BEM** | Методология именования классов (Блок\_\_Элемент--Модификатор) |
| **Pixel Perfect** | Верстка с точным соответствием макету до пикселя |
| **jQuery** | Интерактивность и работа со слайдерами |
| **jQuery Slider** | Слайдер наушников и отзывов |
| **Плавная навигация** | Smooth scroll при переходе по якорным ссылкам |
| **Адаптивная вёрстка** | Корректное отображение на всех устройствах |
| **Flexbox / Grid** | Современная сетка для раскладки элементов |

## ✨ Особенности проекта
 
- 🎯 **BEM-методология** — структурированные и переиспользуемые компоненты
- 🖥️ **Pixel Perfect** — максимальное соответствие дизайн-макету
- 🎠 **jQuery Слайдер** — интерактивные слайдеры для товаров и отзывов
- 🧭 **Плавная навигация** — плавная прокрутка к секциям по якорям
- 📱 **Адаптивность** — корректная вёрстка под мобильные устройства и десктоп
- 🎨 **Секция цветов** — интерактивный выбор цвета наушников
- 💬 **Слайдер отзывов** — карусель с отзывами покупателей
- 👥 **Раздел команды** — слайдер с карточками сотрудников
- 📦 **Форма заказа** — модальное окно с формой доставки
- 📹 **Видео-блок** — секция «Как мы работаем»
